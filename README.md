TaskFlow is a full-stack task management application that allows users to register, log in, and manage their personal tasks. Each user can create, update, delete, and mark tasks as completed or pending.

## Features

- User authentication (Register & Login)
- JWT-based secure authentication
- Create tasks with title and description
- Edit tasks
- Delete tasks
- Mark tasks as Completed / Pending
- User-specific task access
- Modern full-screen UI
